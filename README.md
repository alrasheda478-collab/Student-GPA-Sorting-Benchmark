# Student GPA Sorting Benchmark

## Description
This project compares the performance of Insertion Sort, Merge Sort, and Quick Sort using student GPA records.

## Features
- Sorts Student objects by GPA
- Benchmarks three sorting algorithms
- Tests random, sorted, and reverse sorted datasets.
- Measures execution time
- Counts comparisons
- Counts data movements
- Includes a Java Swing GUI for benchmarking

## Technologies
- Java
- Swing
- IntelliJ IDEA
